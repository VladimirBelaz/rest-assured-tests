# Petstore API Tests

Автотесты для тестирования \[Swagger Petstore API](https://petstore.swagger.io/) с использованием Java, Rest Assured и JUnit 5.

## Цель проекта
Выполнение домашнего задания по автоматизации тестирования API с использованием Rest Assured.

## Технологи
- Java 17
- Maven
- Rest Assured 5.4.0
- JUnit 5 (Jupiter)
- Jackson (для сериализации/десериализации JSON)
- Hamcrest (для гибких проверок)


## Что тестируется
### Метод 1: `GET /pet/{petId}`
- **Тест 1:** Успешное получение существующего питомца (статус 200, проверка полей).
- **Тест 2:** Обработка ситуации с несуществующим ID (статус 404).

### Метод 2: `POST /pet`
- **Тест 1:** Создание нового питомца с минимальным набором обязательных полей.
- **Тест 2:** Создание питомца со всеми возможными полями (категория, теги).


**Всего реализовано 4 автотеста.**
