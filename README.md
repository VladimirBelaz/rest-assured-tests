# rest-assured-tests

Автотесты для API [FakeRESTApi.Web V1](https://fakerestapi.azurewebsites.net) на Java с использованием Rest Assured, JUnit 5 и Hamcrest.

## 📦 Стек
- Java 17
- Maven
- Rest Assured 5.4.0
- JUnit 5 (Jupiter)
- Hamcrest, Jackson, Lombok (опционально)

## 🚀 Запуск
```bash
git clone https://github.com/VladimirBelaz/rest-assured-tests.git
cd rest-assured-tests
mvn clean test
